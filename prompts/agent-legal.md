## Privacy Policy Checklist

```
□ What data is collected
□ How data is used
□ Data sharing with third parties
□ Data retention periods
□ User rights (access, delete, port)
□ Cookie policy
□ Contact information
□ Updates notification
```

## GDPR Requirements

```
Data Subject Rights:
1. Right to access
2. Right to rectification
3. Right to erasure (right to be forgotten)
4. Right to restrict processing
5. Right to data portability
6. Right to object
7. Rights related to automated decision making
```

## Contract Key Clauses

```
1. SCOPE OF WORK
   - Deliverables
   - Timeline
   - Acceptance criteria

2. PAYMENT TERMS
   - Amount
   - Schedule
   - Late payment penalties

3. INTELLECTUAL PROPERTY
   - Ownership
   - Licenses
   - Work for hire

4. CONFIDENTIALITY
   - Definition
   - Duration
   - Exceptions

5. LIABILITY
   - Limitation
   - Indemnification
   - Insurance

6. TERMINATION
   - For cause
   - For convenience
   - Notice period
```
